# Russian Rhymes

**Оффлайн-рифмовник русского языка** для [Obsidian](https://obsidian.md): рифмы, ударения, значения, синонимы и не только — для выделенного слова. Для тех, кто пишет тексты, стихи и рэп на русском.

English: [README.md](README.md)

## Возможности

- **Рифмы по звучанию, а не по написанию** — фонетический рифм-ключ (нормализация хвоста от ударной гласной: редукция, оглушение, кластеры согласных) группирует слова так, как они звучат. Четыре шкалы: точные · близкие · созвучия · ассонансы, плюс аллитерация.
- **Слежение за курсором** — панель сама показывает рифмы к последнему слову строки, которую вы пишете; недописанные и незнакомые слова пропускаются, поэтому выдача не мигает.
- **Ударения** — слово показано с ударением; клик по гласной переносит ударение и пересчитывает рифмы. Ручной выбор запоминается.
- **Вкладка «Значение»** — толкования с примерами и этимологией (из русского Викисловаря) + сворачиваемая таблица форм слова.
- **Вкладка «Ассоциации»** — синонимы, антонимы, гиперонимы/гипонимы, родственные слова, устойчивые сочетания, пословицы, ассоциации, метаграммы и анаграммы.
- **Фильтр «по смыслу»** — подсветить или оставить только рифмы, близкие по значению.
- **Полностью оффлайн** — после разовой загрузки словаря всё работает без интернета.
- **Личные словари** — импорт своих `.dsl` / `.dsl.dz` (Lingvo/GoldenDict) с изменением порядка.
- **Работает на мобильном и десктопе.**

## Как пользоваться

- Выделите слово и нажмите **Ctrl+C дважды** подряд (окно двойного нажатия настраивается; 0 — выключить).
- Или откройте панель с ленты (иконка-перо) / из палитры команд и введите слово.
- Или правый клик по слову в редакторе → **Рифмы к «…»**.
- Или включите **слежение за курсором** (кнопка-прицел справа от поля поиска, команда или настройка) — и просто пишите: рифмы к концу строки появляются сами.

В панели: одиночный клик по слову — скопировать, двойной — провалиться в его рифмы. `Ctrl + ← / →` (или горизонтальный свайп на телефоне) листает разделы.

## Словарь (разовая загрузка)

Obsidian ставит только код плагина (`main.js`, `manifest.json`, `styles.css`). Сам словарь (~73 МБ) **скачивается при первом использовании** из релиза этого репозитория:

- Откройте панель без словаря и нажмите **Скачать словарь (~72 МБ)** — с индикатором прогресса.
- Файлы хранятся в папке плагина и дальше никуда не отправляются.
- Адрес загрузки можно поменять в настройках (если хотите свой хостинг).

Загрузка — явный ручной шаг, чтобы не тратить мобильный трафик и место без вашего согласия.

## Установка

**Из каталога Community Plugins** (после принятия): Настройки → Сторонние плагины → Обзор → «Russian Rhymes» → Установить → Включить.

**Вручную:** скачайте `main.js`, `manifest.json`, `styles.css` из [последнего релиза](https://github.com/mrrepac/russian-rhymes/releases) в `<хранилище>/.obsidian/plugins/russian-rhymes/` и включите плагин.
